# LinkLocker

A bookmark manager monorepo for a CBSD course project.

## Structure

- `apps/cli` - CLI app (empty scaffold for now)
- `packages/database` - encapsulated SQLite database component

## Quick Start

1. Install dependencies:
   - `npm.cmd install`
2. Run database smoke test:
   - `npm.cmd run test -w @linklocker/database`
